# OctaveDB Site

Official website for [OctaveDB](https://github.com/jarednand/octavedb) - a TypeScript-first JSON database for local development.

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
